Question-1: What are the different ways to select an element in the DOM?

--------------------------------------------------------------------------------------------------------------------------------------------

In JavaScript, there are several ways to select an element in the DOM. Each method is used depending on what you want to select.

  • 1.By ID: document.getElementById("idName")
  • 2.By Class Name: document.getElementsByTagName("tagName")
  • 3.By Tag Name:document.getElementsByTagName("tagName")
  • 4.By Name Attribute: document.getElementsByName("nameValue")
  • 5.Query Selector (CSS Selector): document.querySelectorAll("selector")
  • 6.Query Selector All (CSS Selector): document.querySelectorAll("selector")

Question-2: What is the difference between innerHTML, innerText, and textContent ?

--------------------------------------------------------------------------------------------------------------------------------------------

These three (innerHTML, innerText, textContent) are confusing element in the DOM

  • 1.innerHTML: It can add HTML elements dynamically.
  • 2.innerText: It sets the visible text only.
  • 3.textContent: It sets all text inside an element. It doesn't care about CSS visibility.

--------------------------------------------------------------------------------------------------------------------------------------------

Event delegation is a very important DOM concept in JavaScript.

  • 1.What is Event Delegation: Event Delegation is a technique of handling events efficiently.
  • 2.Event Delegation: Putting an event listener on a parent element to handle events for all its children
  • 3.Uses event bubbling mechanism.:
  • 4.Benefits: Performance, less code, works with dynamic elements.

Question-4: What is event bubbling in the DOM?

--------------------------------------------------------------------------------------------------------------------------------------------

Event bubbling is a mechanism in the DOM where an event starts from the target element.

  • 1.Visual Diagram: Child → Parent → Body → Document → Window
  • 2.Why is Bubbling Important: It allows event delegation. Saves performance by reducing number of event listeners.
  • 3.How to Stop Bubbling: event.stopPropagation();
